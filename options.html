LinkedIn Invite Automation Settings

Hotkeys

Use these keyboard shortcuts to control the extension:

  • Alt/Option + W - Fill and send connection request
  • Alt/Option + Q - Fill message only (don't send)
  • Alt/Option + S - Open settings

Note: On macOS, you can use either the Option (⌥) or Command (⌘) key.

Personal Information

Message Template

Available placeholders:

  • {{firstName}} - Connection's first name
  • {{myName}} - Your name
  • {{myRole}} - Your role
  • {{myCompany}} - Your company
  • {{targetRole}} - Target role
